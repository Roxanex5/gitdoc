Qu'es ce que Github ? :

Git est un logiciel de gestion de versions décentralisé.
C'est un logiciel libre créé par Linus Torvalds en 2005, il est en licence libre

Versionner son code est le mécanisme qui conserve la version du code antérieurs de facon a pouvoir le retrouver facilement, meme apres la mise en place de version plus récente
Git hub permet d'avoir une trace des modifications, et de revenir a la version précédente quand la nouvelle version ne convient pas.
Pour utiliser git hub, il n'y as pas besoin de connexions internet, se qui rend son utilisation rapide.
Mais quand on utilise git hub en local on ne peut pas collaborer avec d'autres utilisateurs et il n'y as pas de backup.
C'est pour cela qu'il faut déposer son code sur "https://github.com/."
